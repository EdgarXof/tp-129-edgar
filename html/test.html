1.Explain the difference between a synchronous and asynchronous HTTP request.

A request synchronous block the client until the end of the operation. And a request asynchronous doesn't block the client so the client can done multiple operations.

2.Explain the fundamental difference between 3 web browsers and explain where can we find their compatibility CSS.

Firefox: Open source, supports multiple search engines, using Gecko rendering engine, Consolidates all tabs into a single process, has a wide selection of Add-ons.
Safari: Proprietary(Not Open source), using the Webkit rendering engine.
Google Chrome: More space efficient, Create a process for each tab.
Summary: Firefox is an open source web browser whereas Safari and Google Chrome isn't. Firefox using Gecko for engine layouts whereas Chrome and safari use Webkit engine layouts.
For the CSS comptability we can use this website : caniuse.com
